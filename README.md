# Syd Wells Gambian Project

Official website for the Syd Wells Gambian Football Project - transforming lives in The Gambia through clean water, education, and community development.

## About

For over 30 years, we've been supporting communities in The Gambia through:
- Installing solar-powered boreholes for clean water access
- Building and renovating schools
- Coaching 1000+ children through football and educational programs
- Providing teacher accommodation and school infrastructure
